使用Hexo+GitHub搭建个人博客（一）

  • Hexo

1.安装git
Git-2.11.0.3-32-bit.exe
Git-2.11.0.3-64-bit.exe
选择下载对应自己电脑的版本即可

2.安装nodejs
node-v4.2.3-x86.msi
node-v4.2.3-x64.msi
安装完成后，打开cmd命令输入
git --version
node -v
npm -v
可以依次查看到git，nodejs和npm的版本

3.安装hexo
上述操作无误的话，使用npm命令下载安装hexo，打开cmd输入
npm install hexo-cli -g
安装完成后，输入
hexo -v
检测是否安装成功

4.创建GitHub Pages仓库
全部安装完成后，就需要在github创建远程库
注册登录后点击“+”号，选择New repository
然后在Repository name中写入username.github.io(username是你的github用户名，必须完全一致)
最后Create repository就创建完成了
现在我们就可以在浏览器输入username.github.io访问网站
当然，现在会出现404的页面，因为我们还没有配置页面

5.创建hexo工程
在电脑中创建一个空的文件夹，用来hexo工程
右键这个文件夹，选择Git Bash Here，git窗口启动后输入
npm install hexo --save
然后初始化hexo，依次输入
hexo init
npm install
hexo g
执行完成后，hexo就初始化完毕了，我们可以输入
hexo s
启动一个本地服务器，打开浏览器输入http://localhost:4000/就可以看到我们现在的博客页面

6.把hexo上传到github
接下来把我们本地的hexo项目上传到github我们就可以通过username.github.io访问到我们的个人博客了
要怎么做呢？首先还是在hexo文件夹下Git Bash Here，输入
hexo clean && hexo g && hexo d
清空并生成最新的我们需要上传的文件
然后我们需要下载一个GitHub for windows,然后登录github账号
登陆后选择clone repository，然后把我们之前在github创建的username.github.io仓库clone到本地
现在clone下来的仓库应该是空的，因为我们没有上传任何东西
然后我们把hexo文件夹下的public中的所有文件和文件夹复制粘贴到我们clone下的仓库目录中
然后打开GitHub for windows，会发现Changes里面会有我们刚刚复制进来的文件和文件夹
接下来在下面的Summary输入内容，点击Commit to master提交本地
最后点击Fetch origin按钮把我们刚刚提交的内容同步到github仓库中
现在我们打开github查看是否有我们刚刚上传的文件
如果没有问题的话，打开浏览器输入username.github.io就可以看到我们的个人博客了
